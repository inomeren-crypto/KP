Перечень протоколов и стандартов с обоснованием:
ONVIF Profile C — для передачи событий СКУД в VMS (поддержка в Milestone XProtect и HID).
OSDP Secure — вместо Wiegand, для защищённой связи считыватель ↔ контроллер (требование безопасности самой системы, п. 8.5).
RTSPS — шифрованный видеопоток от камер к серверу.
TLS 1.2+ — для всех API и веб-интерфейсов.
