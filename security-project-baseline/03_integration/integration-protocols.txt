Название сценария: «Неуспешная попытка доступа в архив»
Триггер: событие СКУД — отказ в доступе (биометрия не распознана).
Действие: автоматическая запись 30-сек видеофрагмента со всех 3 камер (15 сек до / 15 сек после).
Протокол: ONVIF Profile C + REST API Milestone.
Ответственный: система (автоматически).
Название сценария: «Аварийное открытие дверей (пожар)»
Триггер: сигнал от пожарной сигнализации → релейный выход в контроллер СКУД.
Действие: все двери разблокируются, СВН переходит в режим полной записи.
Протокол: релейный сигнал + OSDP (для логирования).
